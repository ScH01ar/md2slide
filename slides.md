# 💡 C语言指针：内存地址的“遥控器”

C语言的**指针 (Pointer)** 是其强大和灵活性的核心，但也是许多初学者的难点。

---
title: "1. 什么是指针？"
---

## 1. 什么是指针？

指针本质上是一个**变量**，它存储的不是数据本身，而是**另一个变量在内存中的地址**。

> 📌 **可以把它想象成一个房子的地址牌。**
> * 普通变量：房子本身（存储数据）。
> * 指针变量：地址牌（存储房子的地址）。

---
title: "2. 核心概念与符号"
---

## 2. 核心概念与符号

| 概念/符号        | 含义         | 解释                                                         |
| :--------------- | :----------- | :----------------------------------------------------------- |
| `int *p;`        | **声明**     | 声明一个名为 `p` 的指针变量，它“指向”一个整数 (`int`)。      |
| `&` **取地址符** | **获取地址** | 用来获取变量的内存地址。例：`p = &x;`（将变量 `x` 的地址存入指针 `p`）。 |
| `*` **解引用符** | **访问数据** | 用来访问指针所指向的地址中存储的实际数据。例：`*p = 10;`（通过指针 `p`，将 `x` 的值改为 10）。 |

---
title: "3. 示例代码 (I)"
---

## 3. 示例代码 (I)

```c
#include <stdio.h>

int main() {
    int x = 42;       // 1. 定义一个整数变量 x
    int *p;           // 2. 声明一个指向整数的指针 p

    p = &x;           // 3. 将 x 的地址赋给指针 p
```

---
title: "3. 示例代码 (II)"
---

## 3. 示例代码 (II)

```c
    printf("x 的值是: %d\n", x);          // 输出: 42
    printf("x 的地址是: %p\n", &x);
    printf("p 存储的地址是: %p\n", p);
    printf("p 指向的值是: %d\n", *p);     // 4. 解引用，访问 x 的值

    *p = 100;         // 5. 通过指针修改 x 的值

    printf("x 现在的值是: %d\n", x);      // 输出: 100

    return 0;
}
```
